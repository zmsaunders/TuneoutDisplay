# Assembly Instructions

These can all be printed without supports. Lay the grill and frame on their faces, and the body on its back.

The frame and grill each have a support piece that can be glued to one another. No it's not the most elegant solution, but this was my first leap into CAD. It seemed like the quickest and easiest path forward.

You can drill out the studs for the Mic board a little on the grill if necessary for whatever screws you use. Use something small in the top and bottom mounting points so that it doesn't bend the grill or frame out of the way, but instead will nest into it. I'm honestly not even sure what size the screws I used are - They were in some no-name assorted kit of hardware.

Use a ribbon cable to go from the pi to the mic array so that you can have them disconnected from one another.
